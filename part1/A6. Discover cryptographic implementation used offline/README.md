# A6. Discover cryptographic implementation used offline

Full-disk encryption uses a cryptographic algorithm to scramble all the data stored on a hard drive or SSD. When the computer is turned off, every file, folder, and system file on the disk is stored as unreadable encrypted data. The user must enter a password or PIN when turning on the device. The computer then uses that password to decrypt the disk and allow normal use.

This works completely offline because the encryption and decryption happen locally on the device, without needing an internet connection. 

This also protects data privacy because even if someone steals the computers, they cannot open any files without the correct password.
